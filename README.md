# TP6-CyS-Keylogger

1) Entramos dentro del proyecto, a la carpeta 'TP6-CyS-Keylogger'
2) Colocamos en 'mails.txt' todos los mails a los que queremos mandar mediante el keylogger
3) Ejecutamos el archivo 'main.py'
    Este iniciara preguntando si se quiere mandar mail y/o escribir en un archivo los registros guardados cada cierto tiempo. El archivo en el que se guardan es 'output.txt'. 
    En caso que ninguna opcion sea aceptada el programa finaliza.
